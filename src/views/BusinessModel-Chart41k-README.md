## 目标

还原截图所示的盈亏平衡趋势图：标题为“当单船成本为 41,000$ 时达到盈亏平衡销量”（英文标题），底部有价格标注“Prices : 10,000$”，底部中央图例包含两条系列：
- Annual breakeven sales volume (units)
- Gross profit margin

图表位于 `BusinessModel.vue` 的 Revenue 模态框内，右下角（`chart4Ref`）。

### 一图多要素
- **图型**：双折线（其中销量折线为面积图），双 Y 轴。
- **配色**：
  - **销量**：青色 `#1ed5e6` 折线 + 自下而上的半透明渐变填充（0.1 → 0.8）。
  - **毛利率**：橙色 `#f5a623` 折线，无面积。
- **坐标轴**：
  - **X 轴（类目轴）**：从 `9.57` 到 `7.57` 的 15 个刻度，等间距；`axisLabel` 字号 8，显示竖向分割线；隐藏轴线与刻度；`boundaryGap: false`。
  - **左 Y 轴（销量）**：仅用于数值映射，隐藏轴线/刻度/标签。
  - **右 Y 轴（毛利率）**：范围 0–100，位置在右侧，隐藏轴线/刻度/标签。
- **网格**：`grid: { top: 50, bottom: 45, left: 40, right: 40 }`。
- **标题**：顶端居中，字号 9，常规字重，英文文案：
  - `When the cost of a single ship is 41,000$, break even sales volume is achieved`
- **图例**：底部居中，字号 8，`itemGap: 20`。
- **数据标签**：两条折线的每个点显示标签；销量显示数值；毛利率显示百分号；字号 8，置于点上方（`position: 'top'`）。
- **符号**：两线均为 `symbol: 'circle'`，`symbolSize: 3`。
- **平滑**：两线均为平滑曲线（`smooth: true`）。
- **底部附注**：左下角绘制文本 `Prices :\n10,000$`（两行），字号 8、加粗、左对齐。

## 关键样式/交互
- **自适应**：渲染后调用 `resize()`，并在窗口 `resize` 事件中统一 `resize()`。
- **视觉**：
  - 销量系列带青色面积渐变：从底部 `rgba(30,213,230,0.1)` 过渡到顶部 `rgba(30,213,230,0.8)`。
  - 毛利率为橙色细线；仅显示竖向分割线，隐藏所有轴线与刻度。
- **标签**：每个点显示；毛利率标签追加 `%`；字号 8；上方显示，轻微上移偏移量。

## ECharts 实现要点（与组件对应）
- **容器引用**：`<div ref="chart4Ref" class="modal-chart"></div>`。
- **初始化与配置**：在 `renderModalCharts()` 中的 `option4`：
  - `yAxis: [{ type: 'value' }, { type: 'value', min: 0, max: 100, position: 'right' }]`
  - **销量系列**：`type: 'line' + areaStyle 渐变 + yAxisIndex: 0`
  - **毛利率系列**：`type: 'line' + yAxisIndex: 1`
  - **标题、图例、网格、graphic 文本**：与截图一致（见上文规格）。
- **自适应**：渲染后 `setTimeout(() => instance.resize(), 50)`；窗口变化时统一 `resize()`。
